# Cap at a maximum

Write `cap_at(value, max)` so that it returns `value` if it's at or
below `max`, and `max` otherwise. Both arguments are `i32`. The
logic is one `if` away.

Write the function the most natural way you can think of. The most
natural way doesn't compile. Read the error carefully before
changing anything; it tells you exactly what's wrong, and the fix is
one keyword in one place.
